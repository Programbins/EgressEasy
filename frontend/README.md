# Frontend

## 1. 역할

이 디렉터리는 EgressEasy 사용자가 직접 상호작용하는 모든 시각적 인터페이스(UI)와 사용자 경험(UX) 관련 코드를 포함합니다. 백엔드 서버로부터 받은 데이터를 사용자에게 효과적으로 시각화하고, 사용자의 입력을 받아 백엔드로 전달하는 역할을 담당합니다.

## 2. 주요 기능

- **캡티브 포털(Captive Portal) UI**
  - 사용자가 Wi-Fi에 처음 접속했을 때 마주하는 초기 화면을 구현합니다.
  - 이 화면을 통해 비상구 위치 및 대피 경로 안내 콘텐츠(영상, 인터랙티브 맵 등)를 사용자에게 보여줍니다.

- **경로 시각화**
  - 백엔드로부터 전달받은 현재 위치(비상구)와 목적지(좌석) 좌표를 기반으로, SVG, Canvas 등을 이용해 지도 위에 최단 대피 경로를 동적으로 그려줍니다.
  - 사용자가 자신의 위치를 쉽게 인지할 수 있도록 지도 위에 반짝이는 효과 등 시각적 요소를 구현합니다.

- **사용자 인터랙션 처리**
  - 좌석 선택, '확인' 버튼 클릭 등 사용자의 모든 입력을 처리합니다.
  - 사용자의 행동에 따라 백엔드 API를 호출하여 데이터를 요청하거나 전송합니다.

- **상태 관리**
  - 사용자의 인증 상태, 안내 확인 여부 등 프론트엔드 애플리케이션의 상태를 관리하여 일관된 사용자 경험을 제공합니다.
